springboot-mall

這是一個使用 Spring Boot 開發的電商後端練習專案。


主要功能：

使用者註冊與登入

商品查詢

訂單建立

基本 CRUD 操作


使用技術：

Java

Spring Boot

MyBatis

MySQL


啟動方式：

建立 MySQL 資料庫

設定 application.properties（或 application.yml）

執行 Spring Boot 主程式或使用 mvn spring-boot:run

可用google的API tester 做後端測試


本專案僅供學習與練習使用。
